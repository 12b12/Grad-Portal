# 🎓 Graduation Alumni Portal

A responsive web application that celebrates graduating students by allowing users to explore graduate information by **graduation year** and **school**. The platform highlights student achievements, college destinations, majors, and personal stories.

## 🌟 Overview

The Graduation Alumni Portal was created to help schools and communities preserve student achievements and create a place where future graduates can be inspired by alumni journeys.

Users can:
- Browse graduates by year and school
- View academic achievements and future plans
- Explore alumni stories
- Share their own graduation journey

## ✨ Features

### 🎓 Graduate Directory
- Filter graduates by:
  - Graduation year
  - School
- Displays:
  - Student name
  - Class year
  - School
  - Rank
  - College
  - Major
  - Honors
  - Personal bio

### 🌙 Dark Mode
- User-friendly dark mode toggle
- Improves accessibility and viewing experience

### 📝 Share Your Story
- Allows graduates to share:
  - Name
  - Graduation year
  - School
  - College
  - Major
  - Personal journey

### 📱 Responsive Design
- Optimized for desktop and mobile screens

## 🚀 How to Run Locally

1. Clone the repository:

```bash

```

2. Open the project folder:

```bash
cd Grad-Portal
```

3. Run the project using VS Code Live Server.

4. Open the website in your browser.
